# Weather App

This application provides weather information for a specified city using the OpenWeatherMap API.

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. `git clone https://github.com/Seckinyuksekbas/Weather-App.git`
2. Open `index.html` to run the application.
